## Prisoner's dilemma

```
+------- Prisoner's-Dilemma ------+
|                                 |
|   Blue\Red   Cooperate  Defect  |
|  -----------------------------  |
|  Cooperate      R\R       S\T   |
|                                 |
|     Defect      T\S       P\P   |
|  -----------------------------  |
|                                 |
|         T > R > P > S           |
|                                 |
+---------------------------------+
```
